## Create Terraform Infrastructure with Docker

## Create main.tf
```
terraform {
  required_providers {
    # We recommend pinning to the specific version of the Docker Provider you're using
    # since new versions are released frequently
    docker = {
      source  = "kreuzwerker/docker"
      version = "2.23.1"
    }
  }
}

# Configure the docker provider
provider "docker" {
}

# Create a docker image resource
# -> docker pull nginx:latest
resource "docker_image" "nginx" {
  name         = "nginx:latest"
  keep_locally = true
}

# Create a docker container resource
# -> same as 'docker run --name nginx -p8080:80 -d nginx:latest'
resource "docker_container" "nginx" {
  name    = "nginx"
  image   = docker_image.nginx.image_id

  ports {
    external = 8080
    internal = 80
  }
}
```
## Initialize the project
- which downloads a plugin that allows Terraform to interact with Docker.
```
terraform init
```

Provision the NGINX server container with apply. When Terraform asks you to confirm, type yes and press ENTER.

## Apply
```commandline
terraform apply
```

## Verify NGINX instance
Run docker ps to view the NGINX container running in Docker via Terraform.
```commandline
docker ps
```

## Destroy resources
```commandline
terraform destroy
```

You have now provisioned and destroyed an NGINX webserver with Terraform.
